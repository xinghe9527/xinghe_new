# 🎬 网页服务商视频生成使用指南

## 📋 问题诊断

根据你的截图，错误信息是：
```
Exception: 拒交付失败: Invalid argument(s): img2video 类型必须包含 imageUrl 字段
```

**原因**：你选择了"图生视频"（img2video）工具，但没有提供参考图片。

## ✅ 解决方案

### 方案 1：使用"文生视频"（推荐，最简单）

如果你只想用文字提示词生成视频，应该选择"文生视频"工具：

#### 步骤：
1. 打开设置 → API设置 → 视频模型
2. API 服务商：选择"Vidu（网页服务商）"
3. **选择工具：改为"📹 文生视频"**（不是"图生视频"）
4. 选择模型："Vidu Q3"（免费5次）
5. 点击"保存配置"

然后回到视频空间，输入提示词，点击生成即可。

---

### 方案 2：使用"图生视频"（需要提供参考图片）

如果你确实想用图片生成视频，需要先添加参考图片：

#### 步骤：

**1. 配置设置**
- 打开设置 → API设置 → 视频模型
- API 服务商：选择"Vidu（网页服务商）"
- 选择工具："📹 图生视频"
- 选择模型："Vidu Q3"
- 点击"保存配置"

**2. 在视频空间添加参考图片**
- 进入视频空间
- 在界面上找到"添加图片"按钮（应该在提示词输入框附近）
- 点击按钮，选择一张图片
- 图片会显示在界面上

**3. 生成视频**
- 输入提示词（描述你想要的视频效果）
- 点击生成
- 等待视频生成完成

## 🎯 Vidu 工具说明

### 文生视频（text2video）
- **用途**：纯文字提示词生成视频
- **需要**：提示词
- **不需要**：参考图片
- **示例**：`一个赛博朋克风格的女孩在霓虹灯下行走`

### 图生视频（img2video）
- **用途**：基于图片生成视频
- **需要**：提示词 + 参考图片
- **示例**：
  - 图片：一张女孩的照片
  - 提示词：`让她在霓虹灯下行走`

### 参考生视频（ref2video）
- **用途**：基于参考视频生成新视频
- **需要**：提示词 + 参考视频
- **状态**：暂未实现

## 🔍 如何找到"添加图片"按钮

在视频空间界面上，应该有一个类似这样的按钮：

```
┌─────────────────────────────────────┐
│ 视频空间                            │
├─────────────────────────────────────┤
│ [📷 添加图片]  [参数设置...]        │  ← 这里
├─────────────────────────────────────┤
│ 提示词：                            │
│ ┌─────────────────────────────────┐ │
│ │ 输入你的提示词...               │ │
│ └─────────────────────────────────┘ │
├─────────────────────────────────────┤
│ [▶ 生成视频]                        │
└─────────────────────────────────────┘
```

点击"添加图片"按钮后：
- 会打开文件选择对话框
- 选择一张图片（支持 jpg, png 等格式）
- 图片会显示在界面上
- 最多可以添加 9 张图片

## 💡 提示词建议

### 好的提示词示例
```
✅ 一个赛博朋克风格的女孩在霓虹灯下行走，夜晚，城市街道，雨后
✅ 一只可爱的小猫在阳光下打哈欠，慢动作，特写镜头
✅ 海浪拍打着沙滩，日落时分，金色的光芒，宁静的氛围
```

### 不好的提示词示例
```
❌ 女孩（太简单）
❌ 生成一个视频（没有具体内容）
❌ 随便生成点什么（没有明确指示）
```

### 提示词技巧
1. **描述主体**：人物、动物、物体
2. **描述动作**：行走、奔跑、飞翔
3. **描述环境**：城市、森林、海边
4. **描述氛围**：夜晚、日落、雨天
5. **描述风格**：赛博朋克、写实、卡通
