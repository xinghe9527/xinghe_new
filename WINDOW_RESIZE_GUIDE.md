# 窗口调整大小功能说明

## ✅ 已启用调整大小

在 `lib/main.dart` 中已添加：

```dart
WindowOptions windowOptions = const WindowOptions(
  size: Size(1280, 720),           // 默认尺寸
  minimumSize: Size(1000, 600),    // 最小尺寸
  maximumSize: Size(2560, 1440),   // 最大尺寸（4K）
  ...
);

await windowManager.setResizable(true);  // ✅ 启用调整大小
```

---

## 🎯 目前的限制

### 无边框窗口的特性
应用使用**无边框窗口**（frameless），带来以下特点：
- ✅ 美观的自定义标题栏
- ✅ 现代化的界面设计
- ❌ 没有系统原生的调整大小边框

---

## 🔧 解决方案

### 方案A：使用最大化/还原按钮（当前可用）⭐

**操作步骤**：
1. 点击标题栏右上角的 **方框图标**（最大化按钮）
2. 窗口会填满整个屏幕
3. 再次点击可以还原到原始大小

**优点**：
- ✅ 简单易用
- ✅ 已经可以使用
- ✅ 适合大部分使用场景

---

### 方案B：添加可见的调整大小手柄（需要实现）

在窗口右下角添加一个调整大小图标，类似于传统窗口。

**所需工作**：
- 添加调整大小检测组件
- 在窗口四周和角落添加调整区域
- 实现拖动调整逻辑

**复杂度**：中等
**开发时间**：需要额外实现

---

### 方案C：切换为有边框窗口（最简单）

移除无边框设置，使用系统原生窗口。

**修改 `lib/main.dart`**：
```dart
// 注释掉这一行
// await windowManager.setAsFrameless();

// 或者添加有边框模式选项
WindowOptions windowOptions = const WindowOptions(
  titleBarStyle: TitleBarStyle.normal,  // 使用系统标题栏
  ...
);
```

**优点**：
- ✅ 自动获得系统原生的调整大小边框
- ✅ 用户熟悉的操作方式
- ✅ 无需额外代码

**缺点**：
- ❌ 失去自定义标题栏
- ❌ 界面不够美观

---

## 💡 推荐方案

### 短期方案（当前）
使用 **最大化/还原** 功能：
- 需要更大空间 → 点击最大化
- 需要默认大小 → 点击还原
- 简单有效！

---

### 长期方案（如果需要）
实现**自定义调整大小手柄**：

3. 在窗口边缘添加细微的调整区域

## 🎨 窗口尺寸配置

### 当前设置
```dart

最小尺寸: 1000 x 600  (防止过小)
最大尺寸: 2560 x 1440 (4K分辨率)
```

### 如何修改
编辑 `lib/main.dart` 第44-46行：
```dart
WindowOptions windowOptions = const WindowOptions(
  size: Size(1600, 900),        // 改为更大的默认尺寸
  minimumSize: Size(1200, 700), // 改为更大的最小尺寸
  maximumSize: Size(3840, 2160), // 支持4K屏幕
);
```
