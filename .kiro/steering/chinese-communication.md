# 中文交流规则

## 语言偏好
- 当用户使用中文交流时，你必须使用中文回复
- 代码注释使用中文
- 日志输出使用中文
- 错误提示使用中文
- 文档和说明使用中文

## 技术术语
- 保持英文技术术语的原样（如 API、OSS、Base64、Flutter 等）
- 在必要时可以在括号中提供中文解释

## 代码规范
- 变量名、函数名、类名使用英文（遵循编程规范）
- 注释使用中文
- 字符串常量根据用途选择中文或英文

## 示例
```dart
// ✅ 正确：变量名英文，注释中文
final String userName = '张三';  // 用户名称

// ✅ 正确：日志使用中文
_logger.success('视频生成成功', module: '批量空间');

// ✅ 正确：错误提示中文
throw Exception('未配置 OSS 密钥');
```
