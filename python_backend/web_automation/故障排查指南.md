# 🔧 故障排查指南

## 问题 1: 404 Not Found - 任务不存在

### 错误信息
```
INFO: 127.0.0.1:8597 - "GET /api/task/task_20260227_025718_295798 HTTP/1.1" 404 Not Found
```

### 可能的原因

#### 原因 1: Flutter 在查询旧的任务 ID
**症状**: 
- API 服务器刚启动
- Flutter 在查询之前的任务

**解决方案**:
1. **重启 Flutter 应用**
   ```bash
   # 在 Flutter 中按 R 键热重载
   # 或者完全重启应用
   ```

2. **清除 Flutter 缓存**
   - 在 Flutter 应用中清除任务列表
   - 或者删除 SharedPreferences 中的旧任务

#### 原因 2: 任务提交失败
**症状**:
- 点击生成按钮
- 但任务没有被提交到 API 服务器

**解决方案**:
1. **检查 Flutter 日志**
   ```bash
   # 查看 Flutter 控制台输出
   # 看是否有错误信息
   ```

2. **检查网络连接**
   ```bash
   # 测试 API 是否可访问
   curl http://127.0.0.1:8123/health
   ```

3. **运行简单测试**
   ```bash
   python python_backend/web_automation/test_simple.py
   ```

#### 原因 3: API 服务器重启了
**症状**:
- API 服务器重启后，之前的任务丢失
- Flutter 还在查询旧任务

**解决方案**:
- 这是正常的，任务存储在内存中
- 重启 API 服务器会清空所有任务
- 重启 Flutter 应用即可

## 问题 2: 任务提交后没有反应

### 检查步骤

#### 步骤 1: 确认任务已提交
```bash
# 运行简单测试
python python_backend/web_automation/test_simple.py
```

**预期输出**:
```
✅ 任务 ID: task_20260227_025718_295798
✅ 状态: pending
```

#### 步骤 2: 查看 API 服务器日志
在 API 服务器的终端中，应该看到：
```
✅ 任务已受理: task_20260227_025718_295798
📝 平台: vidu
📝 工具: text2video
📝 提示词: 测试提示词
```

如果没有看到，说明任务没有被提交。

#### 步骤 3: 检查浏览器是否启动
任务提交后，应该会自动启动浏览器。

**如果浏览器没有启动**:
1. 检查 Playwright 是否安装
   ```bash
   playwright install chromium
   ```

2. 检查登录状态
   ```bash
   python python_backend/web_automation/init_login.py vidu
   ```

#### 步骤 4: 查看浏览器中的操作
浏览器启动后，观察：
- 是否打开了 Vidu 网站
- 是否填充了提示词
- 是否点击了生成按钮
- 是否遇到验证码

## 问题 4: Flutter 一直显示"等待中"

### 可能的原因

#### 原因 1: Python 脚本没有返回结果
**检查方法**:
1. 查看 API 服务器日志
2. 看是否有错误信息
3. 看浏览器中的操作是否完成

**解决方案**:
- 如果浏览器保持打开，说明遇到了错误
- 查看浏览器中的具体问题
- 手动完成验证或操作

#### 原因 2: 视频生成时间太长
**正常情况**:
- Vidu 生成视频需要 5-10 分钟
- Flutter 会显示"生成中"（50%）

**解决方案**:
- 耐心等待
- 可以在浏览器中查看生成进度

#### 原因 3: 轮询超时
**检查方法**:
查看 Flutter 日志，看是否有超时错误

**解决方案**:
- 增加超时时间（在 `video_space.dart` 中）
- 或者手动在 Vidu 网站上查看结果

## 常见错误代码

### 404 Not Found
- 任务 ID 不存在
- 可能是查询了旧任务
- 或者任务提交失败

### 400 Bad Request
- 请求参数错误
- 检查 payload 格式
- 检查必需字段

### 500 Internal Server Error
- 服务器内部错误
- 查看 API 服务器日志
- 查看详细错误信息
